# Skill Evaluator

Monitors skill execution, evaluates quality and performance, and provides improvement suggestions.

---

## Operations

- `monitor_execution` - Track skill execution metrics
- `evaluate_quality` - Assess skill output quality
- `analyze_performance` - Analyze execution performance
- `suggest_improvements` - Suggest skill improvements
- `apply_improvements` - Apply suggested improvements
- `generate_report` - Generate evaluation reports
- `analyze_trends` - Analyze usage trends
- `detect_patterns` - Detect usage patterns
- `analyze_skill_interactions` - Analyze how skills work together
